# FinalProject2 - The Notepad+++

## 📖 Description
This project is a Java Swing application developed as part of my OOP101 course.  
It demonstrates object-oriented programming concepts and GUI design using Swing.

## 🛠 Features
- Interactive graphical user interface
- Multiple windows and components
- Event handling (buttons, menus, etc.)
- Object-oriented design principles
- File Handling (Opening a text document and editing content)

## 📂 Project Structure
- `src/` → Java source files
- `out/artifacts/` → Compiled JAR file
- `README.md` → Documentation

## 🚀 How to Run
1. Download the latest release from [Releases](https://github.com/JohnLloydCostales/OOP101-Final_Project_2.git).
2. Run the JAR file with:
   ```bash
   java -jar FinalProject2_new.jar
